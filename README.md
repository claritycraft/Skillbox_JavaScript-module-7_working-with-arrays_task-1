### En:
#### Task 1
Task Description
Create a project folder named “Task 1” with the following files:
- index.html
- index.js
- style.css

#### Step-by-Step Instructions
- In index.html, create a basic HTML structure and connect the JavaScript and CSS files.
- Build a web application called “Home Library” with the following functionality:
- Display a list of books from an array on the page.
- Add a button labeled “Add Book”.
When clicked, it should open a prompt() window where the user can enter a book title.
Before adding the book to the array, validate the input.
If the user enters nothing, show an alert() with the message:
“Book title not entered!”
- Add a button labeled “Search”.
When clicked, it should open a prompt() window to enter a book title for search.
If the book is found in the list, highlight it with a different color.
If not found, show an alert() with the message:
“Book not found!”
- Apply minimal CSS styling to the DOM elements.

#### Example program output
The program is set up with a card inserted and an available balance of 500 rubles.
Input: 1000
Output: Transaction declined

### Ru:
#### Задача 1

#### Что нужно сделать
Создайте папку проекта «Задача 1»с файлами:
index.html;
index.js;
style.css.
В файле index.html создайте базовую HTML-разметку, подключите файл скрипта и файл CSS-стилей.
Создайте веб-приложение «Домашняя библиотека»со следующим функционалом:
На странице должен выводиться список книг библиотеки из массива.
Предусмотрите возможность добавления новой книги в массив. Для этого создайте кнопку «Добавить книгу». Ввод названия книги осуществляется в окне prompt(), которое отображается при клике на кнопку добавления. Перед добавлением книги в массив обязательно сделайте проверку ввода. Если пользователь ничего не ввёл, покажите alert() с сообщением: «Название книги не введено!» 
Для поиска книги в списке создайте кнопку «Поиск», при клике на которую отображается окно ввода prompt() c вводом названия для поиска. Найденная книга в списке должна быть выделена цветом. Если книга не найдена, покажите alert() с сообщением: «Книга не найдена!»
Выполните минимальную CSS-стилизацию DOM-элементов.
